About This Project

This is a simple Library Management System with AI assistance. It helps manage books and user data in an easy and organized way.

Key Features

  • View dashboard with total, available, and borrowed books
  • Add, edit, and delete books easily
  • Search books by title, author, or category
  • AI Assistant suggests books and answers basic queries
  • Manage book categories dynamically

Technology Used

  • Backend: Flask (Python)
  • Database: SQLite
  • Frontend: HTML and CSS
  • Templates: Jinja2

The AI Assistant helps users find books and get recommendations by searching through titles, authors, and categories.
