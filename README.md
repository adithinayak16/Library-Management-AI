Library Management with AI
Welcome to my Library Management System with AI! This is a full-stack web application built using Flask, SQLite, and HTML/CSS, designed to manage library books efficiently and make book recommendations using a simple AI assistant.

Features:
User Authentication:
Users can sign up, log in, and log out securely.

Dashboard:
Shows total books, available books, borrowed books, latest added book, and total users.

Book Management:
Add new books
Edit book details
Delete books

Borrow & Return Books:
Easily borrow available books or return borrowed books.

Search Functionality:
Search books by title, author, or category.

AI Assistant:
Ask questions like:
“Explain Java”
“Recommend beginner books”
The assistant searches your library and gives suggestions.

About Page:
Provides information about the system and its features.

Technologies Used
Backend: Flask (Python)
Database: SQLite
Frontend: HTML, CSS
Session Management: Flask sessions for login authentication

Installation and Setup
Clone the repository:
git clone https://github.com/YourUsername/Library-Management-AI.git

Go to the project folder:
cd Library-Management-AI

Install required packages:
Make sure you have Python installed.
pip install flask

Run the application:
python app.py

Open in browser:
Go to http://127.0.0.1:5000/

Demo:
Log in or sign up as a new user.
Explore the dashboard to see book stats.
Add a new book, edit details, borrow/return books.
Use the AI assistant to get book recommendations or explanations.
